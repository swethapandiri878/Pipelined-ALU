# 4-bit Pipelined ALU (Verilog)

## Overview
This project implements a 3-stage pipelined 4-bit ALU in Verilog HDL.

## Pipeline Stages
Stage 1: Input Register  
Stage 2: ALU Execution  
Stage 3: Output Register  

## Supported Operations
- ADD
- SUB
- AND
- OR
- MUL

## Features
- Parameterized Width
- Fully Synthesizable
- Pipelined for improved timing
- FPGA compatible

## Tools Used
- Verilog HDL
- Vivado (Optional FPGA)
